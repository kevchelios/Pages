Crumple

For Crumple background, my inspiration came during my Typography course. I was sitting at my station, and saw a bunch of wadded up thick paper in a waste bin next to me. I've always really liked the texture of paper and just mixed/media overall. I thought this would make a great background.

    Phase 1: The Physical Media

  • First things first: find a piece of blank paper, preferrably ranpak or some sort of filler/wrapping paper from packaging or gifts. I sourced mine from a thriftstore that wrapped a mug in a very thick wrapping paper.
  • I laid the paper as flat as I could on the floor, while I stood over it, and then used Google Drive's scan document feature to create a PDF of the texture. This was very simple to import into Photoshop.

    • Phase 2: Photoshop

  • We'll start by creating a 400 x 400 pixels.
  • Import the texture pdf file and fill size it to fill out the entire canvas
  • If the texture file exceeds the size of the canvas, select the canvas area and use CTRL+J extract a copy of just that area.
  • Next select from the Filter menu Other>offset

    You'll use the following settings:
  • Using the offset filter, will create seams within the image. All you have to do to blend these is use the Marquee tool (M) to create a selection over the seams, right-clicking the selection and then selecting content aware fill.
  • Because of the shadows of the black and white paper, it may be easier to color the paper using a solid color or gradient, allowing you to use a white text for better visibility. For this reason, we'll us a gradient map to give the highlights a blue tint, and the shadows a more, purple hue. Begin in the adjusment layer menu and select Gradient Map. The settings used are as listed:

    Hex code Percentage
    #3A4DD9 37%
    #402546 0%
  • Now that our color is applied to the texture, we can go ahead and save it to our img directory so it's ready to code into the CSS!
